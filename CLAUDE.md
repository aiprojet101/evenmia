# Evenmia - Organisation d'evenements

## Projet
Site vitrine + devis en ligne pour Evenmia, organisatrice d'evenements (mariages, fiancailles, anniversaires, baptemes, seminaires).

## Stack
- Next.js (App Router) + TypeScript + Tailwind CSS 4
- Framer Motion pour animations
- Google Maps API (distance, autocomplete) via API routes serveur
- Stripe (paiement acompte - a venir)
- Telegram/WhatsApp pour notifications

## Design
- Theme clair/lumineux : creme, rose poudre, vert sauge, dore
- Typographie elegante, boutons arrondis
- Mobile-first
